# PPlab5

Small Python practice project with utility functions and `pytest` tests for learning CI/CD.

## Files
- `utils.py` - Arithmetic helpers (`add`, `subtract`, `multiply`, `divide`) and `int_to_binary` conversion.
- `test_utils.py` - Parametrized unit tests for arithmetic helper functions in `utils.py`.
- `test_binary_conversion.py` - Unit tests for `int_to_binary`, including valid inputs and error cases.
